Учебный проект по анализу временных рядов


- `Нетология временные ряды 1.ipynb` — Jupyter Notebook с анализом временных рядов

## Используемые инструменты

- Python 3.x
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib / Seaborn
- Statsmodels

## Что сделано в проекте:

- Предварительный анализ данных
- Визуализация временного ряда
- Проверка на стационарность
- Декомпозиция временного ряда
- Анализ тренда и сезонности
